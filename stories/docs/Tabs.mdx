# Tabs

Switches between sibling panels in the same space вҖ” one panel visible at a time.
Built on `components/ui/tabs.tsx` (Radix `Tabs`), styled through the Apple-language
tokens. Two looks: a filled segmented control (`default`) and an underline (`line`).

- **Tabs** вҖ” the root; `orientation` `horizontal` (default) or `vertical`.
- **TabsList / TabsTrigger** вҖ” the tab strip; `variant` `default` (pill) or `line` (underline).
- **TabsContent** вҖ” the panel; only the active one renders.

## Usage

- **Do** use Tabs for peer views of the same context (Account / Password / Team).
- **Do** keep tab labels short and parallel; 2вҖ“6 tabs reads best.
- **Do** use `line` in dense or content-heavy layouts; `default` for compact toggles.
- **Don't** use Tabs for a sequence/flow вҖ” that's a stepper or wizard.
- **Don't** hardcode colors вҖ” track/trigger map to `--tabs-*` / semantic tokens.

## Token mapping

| Token | Role |
|---|---|
| `--tabs-list-bg` вҶ’ `--muted` | Segmented track (default variant) |
| `--tabs-trigger-fg` вҶ’ `--muted-foreground` | Inactive label |
| `--tabs-trigger-active-bg` вҶ’ `--background` | Active tab pill |
| `--tabs-trigger-active-fg` вҶ’ `--foreground` | Active label |
| `--tabs-line-indicator` вҶ’ `--foreground` | Underline (line variant) |
| `--tabs-focus-ring` вҶ’ `--ring` | Focus ring |
| `--tabs-list-radius` вҶ’ `--radius-lg` | Track corner |

## Accessibility (WCAG 2.2 AA)

- Radix supplies `role="tablist"/"tab"/"tabpanel"`, `aria-selected`, and arrow-key
  navigation with automatic/manual activation.
- The active tab is conveyed by `aria-selected` **and** the fill/underline, not color alone.
- Visible focus ring on triggers вҖ” never removed.
- Each panel is associated with its tab via `aria-controls`/`aria-labelledby` (Radix).
