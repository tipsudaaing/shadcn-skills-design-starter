# Button

Triggers an action or event. The highest-traffic control in the system вҖ” it
anchors forms, dialogs, and toolbars. Built on `components/ui/button.tsx`
(class-variance-authority), styled entirely through the Apple-language design
tokens.

- **Root** вҖ” native `<button>` (or any element via `asChild` + Radix `Slot`).
- **Label** вҖ” `children`. Icons are auto-sized (`size-4`, smaller for compact sizes).
- **Spinner** вҖ” injected before the label when `loading`.

## Usage

- **Do** keep one `default` (primary) button per view; everything else steps down in emphasis.
- **Do** use `loading` for async submits вҖ” it shows a spinner, sets `aria-busy`, and blocks repeat clicks.
- **Do** give icon-only buttons an `aria-label`.
- **Don't** disable the primary action without telling the user why (prefer inline validation).
- **Don't** hardcode colors вҖ” every surface maps to `--button-*` / semantic tokens.

## Token mapping

| Token | Role |
|---|---|
| `--button-default-bg` вҶ’ `--primary` | Default fill (Apple blue `#0071e3`) |
| `--button-default-fg` вҶ’ `--primary-foreground` | Label on fill (white, 4.70:1 AA) |
| `--button-focus-ring` вҶ’ `--ring` | Focus-visible ring |
| `--button-radius` вҶ’ `--radius-lg` | Corner radius (12px) |
| `--button-transition-duration` | 150ms |

## Accessibility (WCAG 2.2 AA)

- Native `<button>` вҶ’ full keyboard support (Enter/Space) for free.
- Visible focus ring (`focus-visible:ring-3 ring-ring/50`) вҖ” never removed.
- `loading` sets `aria-busy` and disables interaction; the spinner is `aria-hidden`.
- Label/fill contrast verified вүҘ 4.5:1 in both light and dark.
- Color is never the only signal (destructive uses a distinct label, not just hue).
