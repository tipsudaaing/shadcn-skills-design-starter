# Item

A flexible horizontal row for lists, settings, and selections вҖ” media, text, and
actions on one line. Built on `components/ui/item.tsx`, styled through the
Apple-language tokens. Use it where a Card would be too heavy and a bare `<li>` too plain.

- **Item** вҖ” the row; variants `default В· outline В· muted`, sizes `default В· sm В· xs`.
- **ItemMedia** вҖ” leading icon/image (`variant="icon" | "image"`).
- **ItemContent / ItemTitle / ItemDescription** вҖ” the text column.
- **ItemActions** вҖ” trailing controls (buttons, switches).
- **ItemGroup / ItemSeparator** вҖ” stack items into a `role="list"`.

## Usage

- **Do** use Items for settings rows, search results, or pickable lists.
- **Do** use `ItemGroup` so assistive tech announces it as a list.
- **Do** make the whole row a link with `asChild` when the row navigates.
- **Don't** overload a row with many actions вҖ” keep one primary trailing control.
- **Don't** hardcode colors вҖ” surfaces/borders map to `--item-*` / semantic tokens.

## Token mapping

| Token | Role |
|---|---|
| `--item-border` вҶ’ `--border` | Outline variant edge |
| `--item-muted-bg` вҶ’ `--muted` | Muted variant surface (50%) |
| `--item-hover-bg` вҶ’ `--muted` | Hover when interactive |
| `--item-focus-ring` вҶ’ `--ring` | Focus ring (interactive rows) |
| `--item-title` вҶ’ `--foreground` | Title |
| `--item-description` вҶ’ `--muted-foreground` | Description (AA) |
| `--item-radius` вҶ’ `--radius-lg` | Row corner |

## Accessibility (WCAG 2.2 AA)

- Wrap stacks in `ItemGroup` (`role="list"`); each Item is a list entry.
- Interactive rows (`asChild` link/button) keep a visible focus ring вҖ” never removed.
- Don't nest interactive controls inside an already-clickable row (nested-interactive trap).
- Title/description contrast verified вүҘ 4.5:1 in both themes.
