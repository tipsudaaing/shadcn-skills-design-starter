# Checkbox & Radio Group

Choice controls. **Checkbox** (`components/ui/checkbox.tsx`) toggles an independent
option; **Radio Group** (`components/ui/radio-group.tsx`) picks exactly one from a
set. Both are Radix primitives styled through the Apple-language tokens and share
the same fill/ring treatment — document them together.

## Anatomy

```
Checkbox                 Radio Group
┌──┐                     ( ) Default
│✓ │  Label              (•) Comfortable   ← one selected
└──┘                     ( ) Compact
```

- **Checkbox** — `size-4` rounded square; checked fills with `--checkbox-checked-bg`.
- **RadioGroupItem** — `size-4` circle; selected shows a centered dot.
- Both pair with a `Label` (`htmlFor` ↔ `id`).

## Usage

- **Do** use a **Checkbox** for independent on/off options (or a single opt-in).
- **Do** use a **Radio Group** when options are mutually exclusive and all should be visible.
- **Do** give every control a `Label` or `aria-label`.
- **Don't** use a single radio — that's a checkbox. Don't use radios for >6 options (use Select).
- **Don't** hardcode colors — fills/rings map to `--checkbox-*` / `--radio-*` / semantic tokens.

## Token mapping

| Token | Role |
|---|---|
| `--checkbox-checked-bg` → `--primary` | Checked fill + border (Apple blue) |
| `--checkbox-checked-fg` → `--primary-foreground` | Check glyph |
| `--checkbox-border` → `--input` | Resting border |
| `--radio-checked-bg` → `--primary` | Selected fill |
| `--radio-dot` → `--primary-foreground` | Selected dot |
| `--checkbox-focus-ring` / `--radio-focus-ring` → `--ring` | Focus ring |
| `--checkbox-invalid-ring` / `--radio-invalid-ring` → `--destructive` | Error ring |

## Accessibility (WCAG 2.2 AA)

- Radix supplies `role`, `aria-checked`, and keyboard support (Space to toggle;
  arrows to move within a radio group).
- Always provide an accessible name via `Label htmlFor` or `aria-label`.
- Visible focus ring on every control — never removed.
- State is conveyed by the check/dot **and** color, not color alone.
- Error uses `aria-invalid` (the destructive ring) plus accompanying text — never color only.
