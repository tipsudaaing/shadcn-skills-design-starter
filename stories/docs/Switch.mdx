# Switch

A control that toggles a single setting on or off, applied immediately (no submit).
Built on `components/ui/switch.tsx` (Radix `Switch`), styled through the
Apple-language design tokens вҖ” a pill track with a sliding thumb.

## Usage

- **Do** use a Switch for an instant on/off setting; use a Checkbox when the choice
  is part of a form the user submits.
- **Do** pair it with a `Label` (`htmlFor` вҶ” `id`) or an `aria-label`.
- **Do** write the label as the thing being turned on ("Airplane mode"), not an action.
- **Don't** require a separate "Save" вҖ” a Switch implies its change took effect.
- **Don't** hardcode colors вҖ” track/thumb map to `--switch-*` / semantic tokens.

## Token mapping

| Token | Role |
|---|---|
| `--switch-track-on` вҶ’ `--primary` | Checked track (Apple blue) |
| `--switch-track-off` вҶ’ `--input` | Unchecked track |
| `--switch-thumb` вҶ’ `--background` | Sliding knob |
| `--switch-focus-ring` вҶ’ `--ring` | Focus-visible ring |
| `--switch-invalid-ring` вҶ’ `--destructive` | Invalid ring |
| `--switch-radius` | Full pill (`9999px`) |
| `--switch-transition-duration` | 150ms (track color + thumb translate) |

## Accessibility (WCAG 2.2 AA)

- Radix gives `role="switch"` + `aria-checked` and full keyboard support (Space/Enter).
- Always provide an accessible name вҖ” `Label htmlFor` or `aria-label`.
- Visible focus ring (`focus-visible:ring-3 ring-ring/50`) вҖ” never removed.
- State is conveyed by thumb position + track color together, not color alone.
- Disabled uses the real attribute (removed from tab order) at reduced, legible opacity.
